## 项目启动

1. 安装必要的模块

   ```bash
   npm i
   ```

2. 配置环境变量

   - 开发环境: 在根目录下新建`.env.development`文件

     ```text
     # 本地开发环墨境变量配置
     VITE_BASE_API=http://localhost:10089
     VITE_BASE_LAN_API=http://10.10.28.19:10089
     ```

   - 生产环境: 新建`.env.production`文件,内容同上

3. 启动

   ```bash
   npm run dev
   ```

4. 打包

   ```bash
   npm run build
   ```
